Teqly

Comparador de dispositivos electrónicos en México. Permite comparar celulares, tablets, monitores, teclados, ratones y audífonos, mostrando precios en pesos mexicanos (MXN) para ayudar a elegir el mejor dispositivo.
Utilizando una base de datos como es mongo y creando asi mi api de dodne se toma la informacion del backend y se utiliza para el desarrollo de las funciones de la pagina web aqui presentado.
